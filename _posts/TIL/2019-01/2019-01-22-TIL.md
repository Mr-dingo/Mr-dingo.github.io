# friend keyword Posting

c++ friend keyword 에 대해서 새롭게 알게된 사실에 대해서 포스팅했다.
핵심은
- friend class SomeClass;
- friend SomeClass;

는 사용법이 살짝 다르다.

위의 예는 class가 declare 되지 않은 경우 declare 해주고

밑은 class 가 무조건 declare 된 상황이여야한다.
